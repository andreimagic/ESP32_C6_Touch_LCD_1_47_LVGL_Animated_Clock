ESP32-C6 Animated Clock

Interactive LVGL UI with games, animations and touch control

🎬 Demo

📸 Gallery

🔧 Hardware

  • ESP32-C6 Touch LCD 1.47 (Waveshare)
  • AXS5106L Touch Controller (I2C)
  • IMU sensor (FastIMU)

⚙️ Firmware Installation

  1. Install Arduino IDE
  2. Add ESP32 board support
  3. Select "ESP32C6 Dev Module"
  4. Upload the firmware

📦 Dependencies

Required Libraries (exact versions)

  • Arduino_GFX_Library 1.12.5
  • lvgl 9.5.0
  • FastIMU 1.2.8

ESP32 Core

  • esp32 by Espressif Systems
  • Board: ESP32C6 Dev Module

Touch Driver

  • esp_lcd_touch_axs5106l.h (included in ESP32 core)

Note: lv_conf.h is already included and configured in this project.

🔌 Pinout Overview

  • Touch Controller: AXS5106L (I2C)
  • Display: controlled via Arduino_GFX
  • IMU: FastIMU library

🎮 Features

  • Animated clock UI
  • Emotion-based animations
  • Multiple game modes
  • Joystick + servo control

📖 User Manual

Boot Modes

  • D5 HIGH → PullTheWire
  • D5 LOW → ToneQuest
  • D2 HIGH → Servo Mode

Controls

  • Touchscreen navigation
  • Joystick input for servo mode

Features

  • Animated emotional states
  • Interactive mini-games
  • Dynamic UI transitions

💼 Engineering Highlights

  • Embedded UI built with LVGL 9
  • Custom animation engine
  • GPIO-based boot mode selection
  • Real-time touch + IMU input handling
