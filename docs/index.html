ESP32-C6 Touch LCD Animated Clock

LVGL-powered animated clock with multiple game modes

🎬 Demo

📸 Gallery

🔧 Hardware

  • ESP32-C6 Touch LCD 1.47 (Waveshare)
  • Capacitive touchscreen
  • Built-in display

⚙️ Firmware Installation (Arduino IDE)

  1. Install Arduino IDE
  2. Add ESP32 board support
  3. Select board: ESP32-C6
  4. Upload project

📦 Dependencies

  • LVGL
  • TFT_eSPI (or display driver used)
  • Touch driver libraries

🎮 Features

  • Animated clock UI
  • Multiple game modes
  • Joystick + servo control
  • Emotion-based animations

📖 User Manual

Navigate using touch / joystick.
Different boot pins activate different modes:

  • D5 HIGH → PullTheWire
  • D5 LOW → ToneQuest
  • D2 HIGH → Servo Mode
