ESP32-C6 Animated Clock

Interactive LVGL UI with games, animations and touch control

🎬 Demo

📸 Gallery

🔧 Hardware

  • ESP32-C6 Touch LCD 1.47 (Waveshare)
  • Capacitive touchscreen
  • Built-in display

⚙️ Firmware Installation (Arduino IDE)

  1. Install Arduino IDE
  2. Add ESP32 board support
  3. Select board: ESP32-C6
  4. Upload project

📦 Dependencies

Board

Install ESP32 board support from Espressif:

  https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json

Libraries (Arduino Library Manager)

  • lvgl
  • TFT_eSPI

Configuration

  • Configure TFT_eSPI for Waveshare 1.47 display
  • Adjust lv_conf.h (color depth, memory)

🎮 Features

  • Animated clock UI
  • Multiple game modes
  • Joystick + servo control
  • Emotion-based animations

📖 User Manual

Navigate using touch / joystick.
Different boot pins activate different modes:

  • D5 HIGH → PullTheWire
  • D5 LOW → ToneQuest
  • D2 HIGH → Servo Mode
